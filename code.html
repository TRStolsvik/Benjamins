The Code for Benjamins is structured in 3 functions. 


                      function getValues() {
                        // Get start and end values from the page.
                        let startValue = document.getElementById('startValue').value;
                        let endValue = document.getElementById('endValue').value;
                    
                        // Convert the strings that come back to integers.
                        startValue = Number.parseInt(startValue);
                        endValue = Number.parseInt(endValue);
                    
                        // Check and verify that the strings are now in fact integers.
                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                          // Call function generateNumbers with the converted values and store the
                          // returned numbers in a variable.
                          const numbers = generateNumbers(startValue, endValue);
                    
                          // Call function displayNumbers to display the returned array of numbers on the page.
                          displayNumbers(numbers);
                        }
                      }
getValues()

The function getValues() is responsible for getting the values from the user's input, these are the startValue and endValue variables specifically.

Since the values we receive are returned as strings from the HTML, we have to parse them to integers, so we parse each value on it's own using the parseInt() function on the variable we declared and populated in the previous step.

Next, in order to always know that we are working with integers, we check if both values are integers using the isInteger() function, and we only proceed if they both come back as true, which the && helps us with.

If they are indeed both true, we continue with the final two steps, which will run in order: calling the generateNumbers() function with the start and end values as it's parameters and storing it's returned value(s) in a variable called numbers, and then finally, calling the displayNumbers() function with the numbers variable as a parameter in order to display the results on the page. 


                      function generateNumbers(startValue, endValue) {
                        // Declare an empty array called numbers
                        let numbers = [];
                    
                        // For-loop through the the range specified in the function getValues.
                        for (let i = startValue; i <= endValue; i++) {
                          // For each iteration, append the result to the array called numbers.
                          numbers.push(i);
                        }
                        // Finally return numbers.
                        return numbers;
                      }
generateNumbers()

In the function generateNumbers() we take in two parameters inside the open and closing parenthesis. In our case they are the startValue and endValue, separated by a comma. Note that these parameters don't have to be named the same as the variables in the getValues() function, but this does help make it easier to read.

Inside the function, we start by declaring an empty array to store our numbers. Here we've called it numbers.

In order to generate an array of numbers from the startValue to the endValue, we use a for-loop. This loop uses an iterator (i) to store the new value on each iteration of the loop. It starts at 0 and will continue running as long as it's value is lower than or equal to the endValue, and the i++ makes it so that it increments by 1 for every time it loops back around.

For each iteration of the loop, we add the current value of i to our array using the push() function on the array.

Finally, we return the complete array of numbers so that we can use this array in our displayNumbers() function. Note that this return statement happens outside the loop, after it is finished running. 


                      function displayNumbers(numbers) {
                        // Declare a variable called templateRows and set it to an empty string ''.
                        let templateRows = '';
                    
                        // Declare a variable called className and set it to an empty string.
                        let className = '';
                    
                        // For-loop through numbers
                        for (let i = 0; i < numbers.length; i++) {
                          let number = numbers[i];
                    
                          // For each even number, set className to even, else set className to odd.
                          if (number % 2 == 0) {
                            className = 'even';
                          } else {
                            className = 'odd';
                          }
                    
                          // Concatenate (+=) each iteration to the variable called templateRows
                          // using string interpolation.
                          templateRows += `<tr><td class="${className}">${number}</td></tr>`;
                        }
                    
                        // Finally, display the results in the html page.
                        document.getElementById('results').innerHTML = templateRows;
                      }
displayNumbers()

The function displayNumbers() takes in the array of numbers that was returned from our previous function generateNumbers(), as a parameter. This will be used to iterate through in a for-loop in order to display the numbers on the page.

Inside the function we start by declaring two empty string variables for use later, templateRows and className.

Next up we have a for-loop that we start at 0 and iterate for the entire length of the numbers array. Like before, we will iterate one step at a time using i++.

Inside the loop and for each iteration of of the loop we declare a new variable named number (singular) and store the current (i) number/iteration of the numbers array in this variable.

Next we use the modulo operator (%) to check if the remainder of our number divided by 2, is equal to zero. If it is, we store a string called even in the className variable, if it isn't (if it has a remainder that isn't zero) we store a string called odd in the className variable. This will later help us to do some styling to the numbers if we want. For our purposes, we've created a css class that styles the numbers with the "even" class name as bold.

After we've checked the number with the modulo operator, we use template literals to append to the variable templateRows, an HTML table row with a cell with the className relevant for that number, as well as the number itself inside the cell.

Finally we append the templateRows variable to the innerHTML of the results ID of the page, so that it is stored inside the table.
